Open.FlyBridge

An Open Source re-write of the projectFLY bridge plugin for X-Plane.
This plugin was created to resolve bug repoorts that we're seeing with the projectFLY bridge plugin installed.

Installation:
1. Delete X-Plane/resources/plugins/projectFLY
2. Download and extract the latest plugin from the Releases tab here on github.
3. Move the Open.FlyBridge folder into X-Plane/resources/plugins/ folder.
